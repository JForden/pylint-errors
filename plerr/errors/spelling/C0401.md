## C0401 (wrong-spelling-in-comment)

### :x: Problematic code:

```python
# prupose of this function is too...
```

### :heavy_check_mark: Correct code:

```python
# purpose of this function is to...
```

### Rationale:

Used when a word in a comment is not spelled correctly.
