## E0303 (invalid-length-returned)

### :x: Problematic code:

```python
class Foo:
    def __len__(self):
        return -1
```

### :heavy_check_mark: Correct code:

```python
class Foo:
    def __len__(self):
        return 0       # when n >= 0
```

### Rationale:

Used when a __bool__ method returns something which is not a bool
