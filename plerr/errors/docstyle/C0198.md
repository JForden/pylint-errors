## C0198 (bad-docstring-quotes)

### :x: Problematic code:

```python
def foo():
    '''He said "goodbye" and left.'''
```

### :heavy_check_mark: Correct code:

```python
def foo():
    """He said "goodbye" and left."""
```

### Rationale:

Used to enclose a string that can contain both single quotes and double quotes.
