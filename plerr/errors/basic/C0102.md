## C0102 (blacklisted-name)

### :x: Problematic code:

```python
def bar():
    print("test")
    pass
```

### :heavy_check_mark: Correct code:

```python
def Print_Function():
    print("test")
    pass
    
```

### Rationale:

This error occurss when a blacklisted name is used as a variable, Here is the full list of Blacklisted varible names:  
foo  
bar  
baz  
toto  
tutu  
tata  
true  
false
