## W0126 (missing-parentheses-for-call-in-test)

### :x: Problematic code:

```python
def foo():
  pass
  
if foo:
  print('Correct')
```

### :heavy_check_mark: Correct code:

```python
def foo():
  pass
  
if foo():
  print('Correct')
```

### Rationale:

Occurs when a conditional statement (if or ternary if) wrongly call
a function due to missing parentheses.
