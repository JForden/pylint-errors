## W0312 (mixed-indentation)

### :x: Problematic code:

```python
def         foo():
print('Hello')
        pass
```

### :heavy_check_mark: Correct code:

```python
def foo():
  print('Hello')
  pass
```

### Rationale:

Used when there are some mixed tabs and spaces in a module.
