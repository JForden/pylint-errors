## C0301 (line-too-long)

### :x: Problematic code:

```python
print('###############################################################################################################')
```

### :heavy_check_mark: Correct code:

```python
print('#' * 120)
```

### Rationale:

Used when one of code line is longer than the recommended number of characters.
