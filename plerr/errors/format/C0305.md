## C0305 (trailing-newlines)

### :x: Problematic code:

```python
def foo():
  pass
  
  
  
  
  
  
```

### :heavy_check_mark: Correct code:

```python
def foo():
  pass
```

### Rationale:

Used when there are trailing blank lines in a file.
